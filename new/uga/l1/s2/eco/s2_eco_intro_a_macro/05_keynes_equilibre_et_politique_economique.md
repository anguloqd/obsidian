# 05 // keynes : équilibre et politique économique

## L'équilibre keynésien

### L'équilibre de sous-emploi

- Puisqu'il existe du chômage, le niveau production/revenu (noté Y) est strictement inférieur au niveau Y de plein emploi, noté YPE.
- Sur la demande, il faut noter :
    - La demande est la protagoniste dans l'approche keynésienne.
    - **Demande** = Consommation + Investissement + Dépenses publiques.
        - D = C + I + G.
    - Le seul ajustement fait sur le marché est la quantité produite par les firmes.
        - Elles utilisent leurs stocks, puis elles ajustent leurs capacités de production.
- Keynes établit un concept d'équilibre, mais l'équilibre keynésien diffère du classique.
    - **Équilibre keynésien** : offre de biens = demande de biens. On se restraint au marché de travail.
    - **Équilibre classique** : offre de biens et travail = demande de biens et travail.
    - On peut atteindre l'équilibre keynésien et avoir du chômage au même temps.
- L'équilibre keynésien s'écrit Y = D.
- Pour la demande, on ignore pour l'instant les dépenses publiques. Donc, D = C + I.
- Le chapitre passé on avait définit la fonction consommation comme C(Y) = cY + C0.
- Mettant tout ensemble, Y = cY + C0 + I.
- Isolant Y, on a Y = (C0 + I)/(1 - c).
- D'autre côté, il existe une quantité de travail employé L* pour le niveau de production Y*.
- D'ici, on déduit que L* n'assure pas l'équilibre du marché de travail.

### Ajustement de l'équilibre

- La quantité de production est determiné par la demande.
- Keynes insiste sur l'investissement: il est volatile et cause des fortes variations sur le niveau d'équilibre.
- Soit k le multiplicateur d'investissement.
- On notera k comme la dérivée partielle de Y par rapport à I. → k = ∂Y/∂I.
- On arrive a k = 1/(1-c).
- Rappelons que c est la propension marginale à consommer. → 0 < c < 1.
- On en déduit que k > 1.
- Donc, si k > 1, ∂I < ∂Y.
- Finalement, l'impact relative d'un changement de l'investissement est plus forte sur la production.
- Particulièrement, plus c est élevé, plus k est élevé.
- Toutefois, rappelons que l'investissement est volatile, donc délicat.
- Ce n'est pas réaliste de manipuler l'investissement pour améliorer l'économie.
- Approfondissons sur k, le multiplicateur d'investissement. Il a deux dimensions :
    - #1: un investissement est une augmentation des capacités productives : ∆I → ∆Y.
    - #2: il est aussi une augmentation des dépenses de l'investisseur.
- Mais notons que k repose sur #2. Si on augmente la production, notre revenu augmente, donc notre consommation augmente (y comprise l'investissement), et on recommence le boucle.
- ∆I → ∆Y (→ ∆M) → ∆C (→ ∆I) → ∆Y (→ ∆M) → ∆C (→ ∆I) → ∆Y (→ ∆M) → ∆C (→ ∆I)...
- Plus simplement, ∆I → ∆Y → ∆C → ∆Y → ∆C → ∆Y → ∆C...
- On en déduit que multiplicateur d'investissement, k, est un multiplicateur de dépense.
- Dès lors, il est possible d’étendre le raisonnement précédent à d’autres dépenses, y compris les dépenses publiques, G.
- On en peut commencer la politique économique.

## Intervention publique

- Dans le classicisme, l'intervention publique n'est pas justifiée.
- Dans le keynésianisme, elle est une spécificité.
- L'action de l'État se compose de deux leviers : dépenses publiques et collecte d'impôts.
- On va redefinir le revenu de consommateur pour compter les deductions : Yd = Y - T.
- La consommation serait donc C(Yd) = cYd + C0.
- On reprend la demande et on inclut G et Yd.
- D = C + I + G = (cYd + C0) + I + G = c(Y - T) + C0 + I + G.
- Après, on cherche l'équilibre keynésien : Y = D.
- On arrive à Y = (C0 - cT + I + G)/(1 - c).
- Considérons des variations sur les dépenses publiques : ∆G.
- Soit ∂Y/∂G le multiplicateur de dépense publique, qui serait 1/(1-c).
- Notons qu'il prend la même forme que le multiplicateur d'investissement.
- Donc, ∆G → ∆Y. Plus c est grand, plus le mult. de D.P. est grand.
    - Séquence : ∆G → ∆Y → ∆C → ∆Y → ∆C...
- **Thèse keynésienne** : de ce point de vue, invest. publique et privé sont complémentaires.
- Si il y a un équilibre de sous-emploi, l'état peut intervenir avec l'investissement public.
- Or, on ne mentionne pas d'où sort le financement de cet investissement.
- Le consommateurs anticipent une reduction de son salaire, donc pour maintenir la consommation presente, ils réduisent leurs consommation future : l'investissement privé.
- Analysons maintenant les taxes : ∆T.
- Soit ∂Y/∂G le mult. fiscale, qui serait (-c)/(1-c).
- Deux détails :
    - ∆T est de signe contraire à ∆Y.
    - Le mult. fiscale est d'effet mineur que le mult. de D.P.
- Séquence : -(∆T) → ∆Y → ∆C → ∆Y → ∆C...
