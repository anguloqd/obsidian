# 03 // l'utilité

- Utilité marginale :
    - delta U / delta x ou delta y.
    - TmS = Um1 / Um2.
        - Pour les Cobb-Douglas : - (c/d) * (y/x)
- En plus :
    - L'utilité est un concept ordinal !
    - La fonction d'utilité juste nous intéresse pour mettre un ordre sur les paniers.
    - L'écart numérique entre panier 1 et panier 2 n'est pas intéressant.
- Les fonctions quasilinéaires :
    - U = f(x) + y.
    - Dans ce cas, la fonction d’utilité est linéaire par rapport au bien 2 mais (en général) non linéaire par rapport au bien 1.
    - Ex: sqrt(x) + y, ln(x) + y
    - Les fonctions d’utilité quasi-linéaires ne sont pas forcément les plus réalistes mais elles sont très faciles à utiliser.
    - Si on augmente U, le coeff. d'utilité, la courbe d'indiff. se déplace vers le haut (axe y ou x2).
- Normale : monotone et convexe.
