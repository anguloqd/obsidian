# 02 // les préférences

### Hypothèses de préférences

- **Complétitude** : deux paniers peuvent toujours être comparés.
- **Réflexivité** : si deux panier sont égaux, nécessairement les deux paniers sont aussi désirables.
- **Transitivité**.
- En plus, si les deux biens sont non-substituts ni non-compléments...
    - **Monotonie** : le plus, le mieux.
    - **Convexité** : les paniers balancés sont plus préférables que les paniers " extrêmes ".

### Formes des courbes d'indiff.

- **Préférences " normales " (Cobb-Douglas)** : y = cte. / x
    - Hyperbole classique.
- **Préférences quasilinéaires** : cte = f(x) + y.
- **Préférences substitutes** : ax + by = cte.
    - Droite vers le bas.
    - Pente : - a / b.
- **Préférences complémentaires** : cte. = min{ax, by}.
    - Forme de "L" avec le coin étant y=x (même quantités pour les biens).

### Taux marginal de substitution

- La TmS c'est juste la dérivé dans un point en particulier.
    - TmS = delta y / delta x.
    - Il est toujours négatif et mesure la propension marginale à payer.
