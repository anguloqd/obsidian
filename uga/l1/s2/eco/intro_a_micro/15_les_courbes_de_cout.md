## 15 // les courbes de coût

### Les coûts moyens

Types de coûts :

- **Coûts variables** : doivent être payés si et seulement si la firme décide de produire
- **Coûts fixes** : doivent être payés que la firme produise ou non (ex.: loyer)

On peut donc écrire : $C(q) = CV(q) + F$

#### Formules des coûts moyens

>[!important] Mesures du coût moyen
>- **Coût moyen** : $CM(q) = \frac{C(q)}{q} = \frac{CV(q)}{q} + \frac{F}{q} = CVM(q) + CFM(q)$
>- **Coût variable moyen** : $CVM(q) = \frac{CV(q)}{q}$
>- **Coût fixe moyen** : $CFM(q) = \frac{F}{q}$

### Les coûts marginaux

**Coût marginal** :

$$
Cm(q) = \frac{dC}{dq}
$$

Pour le cas discret : $Cm(q) = C(q) - C(q-1)$

Comme $C(q) = CV(q) + F$, on a $Cm(q) = \frac{dCV(q)}{dq}$

### Les coûts marginaux et les coûts variables

L'intégrale de 0 à $q$ du coût marginal est égale aux coûts variables jusqu'à $q$ :

$$
CV(q) = \int_{0}^{q} Cm(t) \, dt
$$

>[!important]
>La courbe de coût moyen atteint son minimum lorsque le coût moyen est égal au coût marginal.

#### Relations entre courbes de coût

- Si $Cm(q) < CM(q)$, alors $CM(q)$ est décroissant
- Si $Cm(q) > CM(q)$, alors $CM(q)$ est croissant
- Si $Cm(q) < CVM(q)$, alors $CVM(q)$ est décroissant
- Si $Cm(q) > CVM(q)$, alors $CVM(q)$ est croissant

### Les économies d'échelle

- **Économies d'échelle** : le coût moyen de production diminue quand la quantité produite augmente
- **Déséconomies d'échelle** : le coût moyen de production augmente quand la quantité produite augmente

Causes des économies d'échelle :

1. Spécialisation et division du travail
2. Indivisibilités techniques (machines sous-utilisées)
3. Loi des grands nombres (réduction relative des stocks de sécurité)
4. Facteur géométrique (relation surface-volume)

Causes des déséconomies d'échelle :

1. Problèmes de coordination
2. Problèmes de motivation
3. Problèmes de communication
